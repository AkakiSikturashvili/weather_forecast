# Weather Forecast Web Application

Django-ზე შექმნილი ამინდის პროგნოზის ვებ-აპლიკაცია, რომელიც იყენებს გარე API-ს რეალური მონაცემების მისაღებად და გააჩნია მომხმარებელთა ავტორიზაციის სრული სისტემა.

## 🚀 ფუნქციონალი

* **ამინდის პროგნოზი:** რეალურ დროში მიმდინარე ამინდის მონაცემების ჩვენება.
* **ავტორიზაციის სისტემა:** მომხმარებლის რეგისტრაცია, სისტემაში შესვლა (Login) და გამოსვლა (Logout).
* **თანამედროვე ინტერფეისი:** სუფთა, საპასუხო (Responsive) და ლამაზი დიზაინი Bootstrap-ისა და Custom CSS-ის გამოყენებით.

## 🛠️ ტექნოლოგიური სტეკი

* **Backend:** Python / Django
* **Frontend:** HTML5, CSS3, Bootstrap 5
* **Database:** SQLite (ნაგულისხმევი)
